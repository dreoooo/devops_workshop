# Dreo's DevOps Todo Manager

A modern full-stack Todo CRUD application built with Next.js, Express.js, Prisma ORM, and PostgreSQL.

## Features

- Create Todo
- View Todo List
- Update Todo
- Delete Todo

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI

### Backend
- Express.js
- Prisma ORM

### Database
- PostgreSQL (Neon)

## Setup

Clone repository:

git clone your-repository-url

Install dependencies:

npm install

Run development server:

npm run dev
